#### 🟢 滑动窗口 / 双指针 — ⭐ 8/10 **最强领域**

**题目数：** 8 题 · **方法论深度：** 极高

已系统掌握滑动窗口的通用模板（右扩找条件、左缩减优化），并在 209、1658、1695 中沉淀了可迁移的标准化方法论。1871 甚至将 DP 与滑动窗口结合做了完整的数学推导。

**代表方法论：**
- `209-minimum-size-subarray-sum` — 滑动窗口 5 段式通用模板 + 迁移性表格
- `1695-maximum-erasure-value` — 思维演化路径 + 四句口诀（"右扩寻找条件成立 / 左缩优化目标值 / ans初始化无穷大 / 边界判定最后做"）
- `1871-jump-game-vii` — DP 递推 → 区间查询的完整数学推导

#### 🟢 动态规划-背包 — ⭐ 9/10 **最强领域**

**题目数：** 8 题 · **方法论深度：** 极高

几乎系统刷完了背包问题的所有变体：0/1 背包、完全背包、二维背包、计数型、最优化型、存在型、容斥组合型。对内外层循环的语义（"时间轴 vs 菜单"理论）有深入理解。

**代表方法论：**
- `377-combination-sum-iv` — 内外层循环顺序对排列/组合的影响，完整的决策表
- `416-partition-equal-subset-sum` — 布尔 DP vs 最大价值 DP 的触发条件对比
- `494-target-sum` — 8 条常见 DP 错误分类，从心态到技术的完整反思
- `474-ones-and-zeroes` — 三维→二维优化的分阶段演化
