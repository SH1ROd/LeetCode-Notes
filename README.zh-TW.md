### 各維度詳解

#### 🟢 滑動視窗 / 雙指針 — 8/10
8 題。系統掌握通用模板（右擴找條件、左縮減優化），209、1658、1695 中有標準化方法論，1871 有 DP 轉區間查詢的完整推導。

#### 🟢 動態規劃-背包 — 9/10
8 題。覆蓋 0/1 背包、完全背包、二維背包、計數/最優化/存在型，對內外層循環語義理解深入。377 有排列 vs 組合決策表，416 有布林 DP vs 價值 DP 觸發器，494 有 8 條錯誤反思。

#### 🟡 前綴和 / 差分陣列 — 7/10
7 題。差分陣列有四句七字口訣：差分長度要加一，首位不變後減前。左標原位右加一，前綴求和只看 N。

#### 🟡 DP-區間排程 / 線性DP — 7/10
6 題。區間排程已形成套路（排序 + 二分轉移 + DP），650 展示了暴力→數學優化的四階段思考。

#### 🟡 哈希表 / 字串 — 6/10
7 題。3Sum 的哈希→雙指針演進詳細，但缺少 hard 級複雜處理。

#### 🔴 棧 / 鏈表 — 3/10
4 題。只有最基礎的括號匹配、鏈表合併、相交鏈表。單調棧是面試高頻，尚未覆蓋。

#### 🔴 優先佇列 / 二分 — 2/10
3 題。缺少 Top-K、資料流中位數、旋轉陣列等經典題型。

#### ⚫ 圖 / 樹 / DFS / BFS — 0/10
0 題。最大空白區。
