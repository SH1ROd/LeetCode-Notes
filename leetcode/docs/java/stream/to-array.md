## Stream.toArray 数据流程表

| 步骤  | 流程动作                                     | 输入                      | 输出        | 说明                                              |
| --- | ---------------------------------------- | ----------------------- | --------- | ----------------------------------------------- |
| 1   | 创建 Stream                                | 原始集合或数组                 | Stream 对象 | Stream 对象记录元素来源                                 |
| 2   | 调用 `toArray(IntFunction<T[]> generator)` | Stream 对象 + lambda/方法引用 | 临时数组      | Stream 会计算元素总数 `n` 并调用生成器                       |
| 3   | lambda/方法引用生成数组                          | `size = n`              | 新数组 `T[]` | 例如 `size -> new String[size]` 或 `String[]::new` |
| 4   | 填充数组                                     | Stream 元素               | 数组 `T[]`  | Stream 内部循环，把每个元素依次放入数组                         |
| 5   | 返回数组                                     | 填充好的数组                  | 用户变量      | 用户接收最终数组，例如 `String[] arr`                      |

---

### 关键点

1. **`size` 参数**  
   - 自动传入 Stream 内部计算出的元素总数  
   - 用来创建长度正确的数组  

2. **lambda 与方法引用等效**  
   - `size -> new String[size]` 等效于 `String[]::new`  
   - 编译器根据 `IntFunction<T[]>` 自动匹配  

3. **Stream 不修改原数据**  
   - 原来的数组或集合保持不变  
   - `toArray` 会生成一个新数组  

---

### 可视化理解

```

原始数组/集合
│
▼
Stream对象
│
▼
计算元素总数 n
│
▼
调用生成器（lambda 或 String[]::new）生成数组
│
▼
将元素依次填充数组
│
▼
返回新数组给用户

```
