### **零件加工**

**时间限制:** C/C++语言 6000MS; 其他语言 8000MS

**内存限制:** C/C++语言 65536KB; 其他语言 589824KB

#### **题目描述:**

小Q需要加工两个精密零件。这两个零件分别有 $n$ 个关键位置，分别从 $1$ 到 $n$ 进行编号。在加工结束后，这两个零件最后会组合在一起，并且关键位置一一对应。现在，小Q拿到了经过粗加工后的零件。其中，第一个零件的第 $i$ 个位置的高度为 $a_i$，第二个零件的第 $i$ 个位置的高度为 $b_i$。

小Q现在需要对两个零件进行精细加工。加工结束后，两个零件的所有关键位置在拼接后的高度必须相同。并且为了保证零件强度，精细加工后同一个零件编号相邻的两个位置的高度之差小于等于 $k$。

你能否帮助小Q计算，要使精细加工后的零件满足上述要求，最少需要磨掉多少高度的零件？

形式化的，给定两个数组 $a$ 和 $b$，你需要找到两个数组 $a'$ 和 $b'$，满足以下条件：

1. 对于所有 $1 \le i \le n$，$0 \le a'_i \le a_i$，$0 \le b'_i \le b_i$。
    
2. 存在某个定值 $h$，使得对于所有 $1 \le i \le n$，$a'_i + b'_i = h$。
    
3. 对于所有 $1 \le i < n$，$|a'_{i+1} - a'_i| \le k$。
    

输出 $\sum_{i=1}^{n} ((a_i - a'_i) + (b_i - b'_i))$ 的最小值。

#### **输入描述:**

输入第一行有一个整数 $T$ $(1 \le T \le 10^5)$，代表数据组数。

之后的每一组数据，第一行有两个整数 $n$ 和 $k$ $(2 \le n \le 10^5, 1 \le k \le 10^9)$，含义如题面所述。

接下来 $n$ 行，第 $i$ 行有两个整数 $a_i$ 和 $b_i$ $(1 \le a_i \le 10^9, 1 \le b_i \le 10^9)$，含义如题面所述。

保证所有 $T$ 组输入数据的 $n$ 之和小于等于 $2 \times 10^5$。

#### **输出描述:**

共 $T$ 行，每行一个整数，代表该组数据的答案。

#### **样例输入:**

Plaintext

```
1
4 3
3 6
5 2
4 7
8 9
```

#### **样例输出:**

Plaintext

```
16
```
