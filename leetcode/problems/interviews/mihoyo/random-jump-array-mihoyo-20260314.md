### 题目：乱跳的数组 (hard)

**题目描述**

对于给定的长度为 $n$ 的数组 $\{a_1, a_2, \dots, a_n\}$，我们定义“跳数”为同时严格大于或小于左右相邻数的数字，形式化的说，对于第 $i\ (1 < i < n)$ 个整数 $a_i$，它被称为“跳数”，当且仅当满足 $a_{i-1} < a_i > a_{i+1}$ 或 $a_{i-1} > a_i < a_{i+1}$。

若一个数组中，所有的满足 $i \in (1, n)$ 的数字 $a_i$ 均为“跳数”，且任意相邻的两个元素 $a_j, a_{j+1}\ (1 \le j < n)$ 都不相等，则称该数组为“乱跳的数组”。

现在，对于给定的初始数组，计算最少需要从原数组中删除的数字个数，使得剩余数字按原相对顺序拼接成的数组是一个“乱跳的数组”。

---

**输入描述**

第一行输入一个整数 $n\ (3 \le n \le 2 \times 10^5)$ 代表数组中的元素数量。

第二行输入 $n$ 个整数 $a_1, a_2, \dots, a_n\ (-10^7 \le a_i \le 10^7)$ 代表数组元素。

---

**输出描述**

在一行上输出一个整数，代表最少需要删除的数字个数。

---

### 示例 1

**输入**

Plaintext

```
7
1 3 1 4 5 2 0
```

**输出**

Plaintext

```
2
```

**说明**

其中一种最优的方案是删除数组中的第五、七个数字。

---

### 示例 2

**输入**

Plaintext

```
3
2 2 2
```

**输出**

Plaintext

```
2
```
