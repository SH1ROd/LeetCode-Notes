#动态规划 #组合数学
# 题意解析

我们有一个长度为 `n` 的数组 `a`，满足以下条件：

- **非严格递增：**
  a1 ≤ a2 ≤ ... ≤ an

- **部分元素缺失：**
  某些位置的值丢失了（用 `0` 表示）。

- **边界条件：**
  `a1` 和 `an` 不为 `0`。

---

## 要求

统计所有可能的原始数组的方案数，结果对 **10^9 + 7** 取模。

---

## 样例分析

**样例：**

```

n = 3
a = [1, 0, 5]

```

**可能的方案：**

```

1 1 5
1 2 5
1 3 5
1 4 5
1 5 5

```

**共 5 种方案。**
