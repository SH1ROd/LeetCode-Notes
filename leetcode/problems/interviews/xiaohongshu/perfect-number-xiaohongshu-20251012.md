# 题意解析

我们定义一个正整数 `x` 是 **“完美数字”**，当且仅当它可以写成：

> **长度至少为 3** 且 **公差为 1** 的 **正整数等差数列** 的乘积。

---

## 定义形式

等差数列形如：

```

a, a + 1, a + 2, …, a + k - 1

```

其中：

- `k ≥ 3`
- 所有元素都是正整数

于是有：

```

x = a × (a + 1) × (a + 2) × … × (a + k - 1)

```

其中 `a ≥ 1`。

---

## 示例

```

6  = 1 × 2 × 3        → YES
24 = 2 × 3 × 4        → YES
120 = 4 × 5 × 6       → YES

```

---

## 不是完美数字的例子

```

1  → NO
2  → NO
3  → NO
4  → NO
8  → NO

```
