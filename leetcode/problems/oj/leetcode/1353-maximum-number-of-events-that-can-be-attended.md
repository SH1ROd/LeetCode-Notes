#优先队列
[1353. 最多可以参加的会议数目 - 力扣（LeetCode）](https://leetcode.cn/problems/maximum-number-of-events-that-can-be-attended/description/)
> 半自己写
```c++
class Solution {
public:
    int maxEvents(vector<vector<int>>& events) {
        sort(events.begin(), events.end());
        auto cmp = [](vector<int> &a, vector<int> &b){
            return a[1] > b[1];
        };
        priority_queue<vector<int>, vector<vector<int>>, decltype(cmp)> pq(cmp);
        int n = events.size();
        int curr_event = 0;
        int curr_day = 0;
        int ans = 0;
        while(curr_event<n || !pq.empty()){
            if (pq.empty()) {
                curr_day = events[curr_event][0];
            }
            while(curr_event < n && events[curr_event][0] <= curr_day){
                pq.push(events[curr_event]);
                curr_event++;
            }
            while(!pq.empty() && pq.top()[1] < curr_day){
                pq.pop();
            }
            if(!pq.empty()){
                pq.pop();
                ans++;
                curr_day++;
            }
        }
        return ans;
    }
};
```

> GPT版本
```c++
class Solution {
public:
    int maxEvents(vector<vector<int>>& events) {
        // 先把事件按开始时间从小到大排序，方便我们按时间顺序处理
        sort(events.begin(), events.end());

        // 小顶堆，存放当前“可参加事件”的结束时间
        // 优先参加结束时间最早的事件（堆顶就是最小结束时间）
        priority_queue<int, vector<int>, greater<int>> pq;

        int day = 0;   // 当前时间（天数）
        int i = 0;     // 指向events数组中下一个还没处理的事件
        int n = events.size();
        int ans = 0;   // 记录最多参加事件数量

        // 当还有事件没处理 或 堆中还有可参加事件时，循环继续
        while (!pq.empty() || i < n) {
            // 如果堆为空，说明当前没“可参加”的事件
            // 那么把时间跳到下一个事件的开始时间
            if (pq.empty()) {
                day = events[i][0];
            }

            // 将所有开始时间 <= 当前天的事件的结束时间加入堆中
            // 这些事件都是“当前可参加”的
            while (i < n && events[i][0] <= day) {
                pq.push(events[i][1]);
                i++;
            }

            // 从堆中移除所有已经过期（结束时间 < 当前天）的事件
            // 这些事件已经结束，无法参加了
            while (!pq.empty() && pq.top() < day) {
                pq.pop();
            }

            // 如果堆里还有事件，说明有事件可参加
            if (!pq.empty()) {
                pq.pop();  // 参加结束时间最早的那个事件（弹出堆顶）
                ans++;     // 参加事件数加一
                day++;     // 天数加一，继续处理下一天
            }
        }

        return ans;  // 返回最多参加事件数
    }
};

```

### 💡 总思路：

1. 按开始时间排序。
    
2. 使用小顶堆 `pq` 存“可参加的事件”的结束时间。
    
3. 每天处理：
    
    - 加入所有开始时间 ≤ 当前天 的事件；
        
    - 移除已经过期的事件（结束时间 < 当前天）；
        
    - 从堆顶选择结束时间最早的事件参加。
